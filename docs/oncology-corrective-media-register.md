- Intrinsic view box: 1600 × 900.
- Vector delivery at every viewport without raster upscaling.
- No remote request, font dependency, embedded script or external image reference.
- Descriptive `<title>` and `<desc>` inside each SVG plus page-level `alt` text.
- Lazy-loaded below the first viewport.
- Displayed in a responsive three-, two- and one-column editorial grid.

# Remaining Items

## Requires External Credentials
- Production hosting credentials.
- Microsoft Graph application registration.
- SharePoint site and drive identifiers.
- Production portal password hash variables.
- DNS or deployment platform access to make changes live on the public domain.

## Requires Business Approval
- Final wording for regulated service claims.
- Product portfolio claims before publishing specific medical products.
- Privacy policy and cookie consent approach if analytics is enabled.
- Legal/regulatory review before using regulated claims in paid marketing.

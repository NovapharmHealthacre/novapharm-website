## Owner correction process

For every inconsistent external reference record:

1. URL and platform.
2. Current incorrect value.
3. Canonical replacement value.
4. Verification evidence.
5. Account owner or editorial contact.
6. Request date and status.
7. Date corrected or reason declined.

Do not create extra profiles merely to outnumber an incorrect source. Correct controlled accounts first, then request factual corrections only from legitimate publishers.
